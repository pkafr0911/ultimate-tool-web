# Ultimate Tool

### Development

#### requirement

Yarn or npm

Node > v18 (you can install directly or use nvm)

#### yarn

```
#1st time only
yarn install

yarn start

```

#### npm

```
#1st time only
npm install

npm run start

```

### Deploy

#### requirement

Docker

```
docker build --no-cache -f Dockerfile -t new-edu .
docker compose up
```
